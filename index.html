$ info modemizr

Modemizr is a simple Javascript library that simulates the "old BBS with slow modems" effect on HTML pages.

It can handle arbitrary HTML elements with styling as well as things such as changing output speeds, pausing .... and even supports "slow loading style" images.


Where to get it

$ git clone https://github.com/santtu/modemizr.git
$ open https://github.com/santtu/modemizr

Background

Modemizr was inspired by Asciinema.
